# hero-image-generator

# Live
At this moment it only works locally, you can download this project and turn on the live server.

# Features
You can upload a photo, add text to that and increase or decrease the size, change the background color on the photo. Your photo will be stored in localstorage.

# Technologies Used
• HTML
• SCSS / CSS
• JavaScript
